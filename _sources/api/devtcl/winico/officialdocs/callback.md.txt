# 回调事件
以下 `Windows` 事件将生成对 `winico` 任务栏命令注册的回调过程的调用。下面列出了回调替换中 `%m` 的所有可能值。通常，`WM_RBUTTONUP`绑定到上下文菜单。

## 事件
:::{card} WM_MOUSEMOVE
鼠标移动时触发事件
:::

:::{card} WM_LBUTTONDOWN
鼠标左键按下时触发事件
:::

:::{card} WM_LBUTTONUP
鼠标左键放开时触发事件
:::

:::{card} WM_LBUTTONDBLCLK
鼠标左键双击时触发事件
:::

:::{card} WM_RBUTTONDOWN
鼠标右键按下时触发事件
:::

:::{card} WM_RBUTTONUP
鼠标右键放开时触发事件
:::

:::{card} WM_RBUTTONDBLCLK
鼠标右键双击时触发事件
:::

:::{card} WM_MBUTTONDOWN
鼠标中键按下时触发事件
:::

:::{card} WM_MBUTTONUP
鼠标中键放开时触发事件
:::

:::{card} WM_MBUTTONDBLCLK
鼠标中键双击时触发事件
:::
