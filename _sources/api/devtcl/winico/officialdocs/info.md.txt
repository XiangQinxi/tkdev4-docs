# winico info

:::{card} 参数
`id`
:::

返回有关此扩展管理的图标资源的信息。如果省略`id`，则该命令将返回所有当前图标`ID`的列表。
如果指定了图标`ID`，则结果是图标资源中每个图标的列表选项值对于通过`winico`创建的标识符，加载给出的值不相关。

## 额外选项

### -pos
图标资源内的位置（从 0 开始）

### -width
图标的宽度（以像素为单位）

### -height
图标的高度（以像素为单位）

### -geometry
宽度x高（以像素为单位）（为方便起见）

### -bpp
颜色深度（以每像素位数为单位）（4 表示 16 种颜色）

### -hicon
此图标的windows图标句柄（在实际版本中，从磁盘读取后将立即创建资源的所有hicons，在更高版本中，这可能会按需发生，以节省系统负载）

### -ptr
内部结构的C指针，只对黑客感兴趣：-）
